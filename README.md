# Clean Amazon Search

Amazon.co.jpでのお買い物を、もっと安心に。

国内発送・Amazon配送の商品を優先表示するChrome拡張機能です。日本のメーカーや専門店の商品も含めて、信頼できる商品を簡単に見つけられます。

## 特徴

- **海外発送商品を除外** - 配送遅延・紛失リスクを軽減
- **日本の優良セラーも表示** - パナソニック、東急ハンズなどのメーカー直販・専門店も残る
- **ワンクリックで適用** - 面倒な設定不要
- **3段階のフィルター** - ニーズに合わせて選択可能

## インストール

### 開発版（手動インストール）

1. このリポジトリをダウンロード or クローン
2. Chromeで `chrome://extensions` を開く
3. 右上の「デベロッパーモード」をON
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. `clean-amazon-search` フォルダを選択

## 使い方

### 基本操作

1. Amazon.co.jpで商品を検索
2. ツールバーの拡張機能アイコンをクリック
3. 「安心フィルターを適用」ボタンを押す

フィルターが適用され、国内発送・Amazon配送の商品のみが表示されます。

### フィルターレベル

| レベル | 条件 | おすすめの人 |
|--------|------|--------------|
| **スタンダード** | 国内発送 + Amazon配送（FBA） | 一般ユーザー（推奨） |
| **プレミアム** | 上記 + 評価★4以上 | 品質重視の方 |
| **ストリクト** | Amazon公式販売のみ | 超慎重派の方 |

### 各フィルターで表示される商品

| 商品タイプ | スタンダード | プレミアム | ストリクト |
|------------|:------------:|:----------:|:----------:|
| Amazon直販 | ○ | ○ | ○ |
| 日本メーカー直販（FBA利用） | ○ | ○ | × |
| 国内専門店（FBA利用） | ○ | ○ | × |
| 海外発送 | × | × | × |

## 機能一覧

### メイン機能

- **ワンクリック・フィルター適用** - ポップアップからすぐに適用
- **自動適用モード** - Amazon検索時に自動でフィルター（設定でON/OFF）
- **ページ内ボタン** - 検索結果ページに「安心フィルターで再検索」ボタンを表示

### 追加機能

- **商品詳細ページ警告** - 海外発送商品を開いた際に注意バナーを表示
- **セラー住所判定** - 販売元の住所をチェックして日本/海外を自動判定
- **統計ダッシュボード** - フィルター使用回数を表示
- **キーボードショートカット**
  - `Alt+Shift+A` - フィルター適用
  - `Alt+Shift+S` - 自動適用ON/OFF切り替え

## なぜ「Amazon公式のみ」ではないのか

「Amazon公式のみ」フィルターでは、以下の優良セラーまで除外されてしまいます：

- 日本メーカー直販（パナソニック、ソニー、タイガー魔法瓶など）
- 老舗専門店（東急ハンズ、ヨドバシ、ビックカメラなど）
- 職人・工房（日本製革製品、燕三条の金属製品など）
- 国内中小メーカー

本拡張機能は「**誰が売るか**」ではなく「**どう届くか**」で判断します。国内発送 + Amazon配送なら、日本の優良セラーも含めて安心してお買い物できます。
