## 1. 課題の再定義

### ユーザーの本当の課題

> 「Amazonで商品を探すとき、同じような商品で商標だけが変わっている転売屋の商品が大量に表示されるのが嫌。これをなくしたい。」

### 従来のアプローチの問題

| 従来 | 問題点 |
|------|--------|
| 海外発送を避ける | 転売品でも国内発送のものがある |
| FBA優先 | FBAを使っている転売屋も多い |
| Amazon公式のみ | 良い日本のセラーまで消える |

**結論**: 従来のアプローチは問題の本質を捉えていなかった

## 2. 転売品・OEM品の特徴分析

### 典型的な検索結果の問題

```
「スマホケース」で検索:
1. 【BrandA】スマホケース クリア... ¥999
2. 【BrandB】スマホケース クリア... ¥1,099
3. 【BrandC】スマホケース クリア... ¥899
4. 【BrandD】スマホケース クリア... ¥1,199
5. 【BrandE】スマホケース クリア... ¥999
   ↑ 全部同じ商品、ブランド名だけ違う
```

### 転売品・OEM品の共通特徴

| カテゴリ | 特徴 | 検出難易度 |
|---------|------|-----------|
| **ブランド名** | 聞いたことがない、アルファベット羅列、日本語として不自然 | ★★☆ |
| **商品画像** | 複数出品で同じ画像を使用、白背景の簡素な画像 | ★★★ |
| **価格帯** | 異常に安い、または相場と比べて不自然 | ★★☆ |
| **レビュー** | 短期間に大量の高評価、日本語が不自然 | ★★★ |
| **商品説明** | 機械翻訳っぽい、詳細がない | ★★☆ |
| **セラー情報** | 新規セラー、評価が少ない、海外住所 | ★★☆ |
| **商品バリエーション** | 同じ商品が大量のカラバリで出品 | ★☆☆ |

## 3. 技術的アプローチの検討

### 案A: ブランドホワイトリスト（却下）

- 確実だが維持が大変
- 新しい良いブランドを見逃す
- カテゴリごとに違う

### 案B: ネガティブスコアリング（採用）

```
怪しさスコア =
  + ブランド名が怪しい（+20〜30点）
  + セラー評価が低い（+20点）
  + 海外セラー（+20点）
  + セラーが新規（+15点）
  + ノーブランド表記（+15点）

閾値: 40点以上 → 非表示 or 警告
```

### 案C: 重複商品検出（将来実装）

- 商品タイトルの類似度計算
- 同一商品グループから最も評価の高いものだけ表示
- 技術的に複雑なためv2.0以降

### 案D: ハイブリッド（採用）

- 第1層: URLパラメータ（Prime、評価）
- 第2層: 独自判定（ブランド名、セラー情報）

## 4. 新しいフィルタリング体系

### コンセプト変更

| 従来 | 新 |
|------|-----|
| 「海外発送を避ける」 | 「転売品・OEM品を排除して本物を見つける」 |
| 「Amazon配送優先」 | 「信頼できるブランド・セラーを優先」 |

### 新フィルターレベル

| レベル | 名称 | 条件 | 残る商品 |
|--------|------|------|----------|
| **Level 1** | ベーシック | Prime対象 | 配送が安心な商品 |
| **Level 2** | スタンダード | Prime + 評価★4以上 | 評価の高い商品 |
| **Level 3** | プレミアム | 上記 + 怪しいブランド除外 | 信頼できそうな商品 |
| **Level 4** | ストリクト | 上記 + Amazon公式のみ | Amazon直販のみ |

### URLパラメータ

| レベル | パラメータ |
|--------|-----------|
| ベーシック | `p_85:2322926051` (Prime) |
| スタンダード | 上記 + `p_72:83461051` (★4以上) |
| プレミアム | 上記 + 独自判定で非表示処理 |
| ストリクト | `p_6:AN1VRQENFRJN5` (Amazon公式) |

---

## 5. 怪しいブランド判定基準

### スコアリング方式

| 条件 | スコア | 例 |
|------|--------|-----|
| ブランド名がアルファベット羅列 | +20 | 「XYZABC」「QWERTY」 |
| ブランド名が極端に短い（1-2文字） | +15 | 「A」「XY」 |
| ブランド名が極端に長い（15文字以上） | +10 | 「ABCDEFGHIJKLMN」 |
| ノーブランド表記 | +15 | 「ノーブランド品」「Generic」 |
| セラー評価90%未満 | +20 | - |
| セラーが新規（6ヶ月未満） | +15 | - |
| 海外セラー（住所判定） | +20 | 中国、香港など |
| レビュー数が極端に少ない（10件未満） | +10 | - |

### 判定閾値

| スコア | 判定 | 動作 |
|--------|------|------|
| 0-19 | 安全 | そのまま表示 |
| 20-39 | 注意 | 警告バッジ表示 |
| 40以上 | 危険 | 非表示（設定で表示可） |

## 8. 最終決定事項まとめ

| 項目 | 決定内容 |
|------|----------|
| **コンセプト** | 「転売品・OEM品を排除して本物を見つける」 |
| **フィルター体系** | 4段階（ベーシック→スタンダード→プレミアム→ストリクト） |
| **デフォルト設定** | スタンダード（Prime + ★4以上） |
| **怪しいブランド判定** | スコアリング方式（閾値40点で非表示） |
| **非表示機能** | 有効化（怪しい商品を自動非表示） |
| **ユーザー選択** | 非表示商品を「すべて表示」で確認可能 |
| **統計表示** | 除外件数・表示件数をリアルタイム表示 |

## 9. 前回計画との差分

| 項目 | v2計画（前回） | v3計画（今回） |
|------|----------------|----------------|
| 主な目的 | 海外発送を避ける | 転売品・OEM品を排除 |
| フィルター基準 | 配送条件（FBA、国内発送） | 信頼性スコア（ブランド、セラー） |
| 判定対象 | セラーの住所 | ブランド名 + セラー情報 + 住所 |
| 非表示機能 | バナー表示のみ | 自動非表示 + 件数表示 |
| 最終目標 | 配送トラブル回避 | 本当に良い商品を見つける |
