# Amazon 転売品/OEM品パターン調査レポート

## 調査日: 2026-02-01

---

## 1. 調査概要

### 対象カテゴリ
1. スマホスタンド
2. モバイルバッテリー
3. ワイヤレスイヤホン
4. USB充電器
5. LEDライト

### 調査方法
- Web検索による情報収集
- 既存の見分け方記事・ブログの分析
- 現在のClean Amazon Searchフィルタリングロジックとの照合

## 2. 転売品/OEM品の特徴パターン

### 2.1 ブランド名の特徴

#### 怪しいブランド名のパターン

| パターン | 例 | 検出難易度 |
|---------|---|-----------|
| ランダム英数字 | HXCQYGL, WYSWYG, LYHO | 中 |
| 意味不明な造語 | Bennbari, Simpolu, Freeberty, GiliMax, Cuicycric | 中 |
| 有名ブランド模倣 | Proscenic, Potensic（Panasonicに類似）| 高 |
| JP/日本偽装 | STSEETOP JP, ○○ JP | 低 |
| 短すぎる英字 | MITSOLY, OSLEEP, LAONID | 中 |

#### 信頼できるブランド名

| カテゴリ | 信頼できるブランド |
|---------|------------------|
| モバイルバッテリー | Anker, Belkin, cheero, RAVPower, Panasonic |
| ワイヤレスイヤホン | Sony, Bose, JBL, Jabra, Apple, Beats |
| USB充電器 | Anker, Belkin, RAVPower, Elecom, BUFFALO |
| 中華でも信頼可 | SoundPEATS, KZ, Xiaomi, PZX（実績あり）|

### 2.2 商品タイトルの特徴

#### 怪しいタイトルパターン

1. **冒頭の誇張キーワード**
   - 【令和最新版】
   - 【2025新設計】
   - 【業界トップ】
   - 【アップグレード版】
   - 【進化版】
   - 【最強】

2. **スペック羅列**
   - 【Bluetooth5.3技術 蓋を開けたら接続】
   - 【業界トップ Bluetooth5.3+38時間連続再生】
   - 過度に長いタイトル（100文字以上）

3. **不自然な日本語**
   - 機械翻訳特有の文法
   - 句読点の使い方が不自然

#### 正規タイトル形式
```
ブランド名 + 商品名 + 型番
例: Anker PowerCore 10000 (10000mAh 最小最軽量 大容量 モバイルバッテリー)
```

### 2.3 価格帯の特徴

| カテゴリ | 怪しい価格帯 | 信頼できる価格帯 |
|---------|------------|----------------|
| モバイルバッテリー(25000mAh) | 2,500〜3,000円 | 5,000〜6,000円 |
| ワイヤレスイヤホン | 185〜1,500円 | 3,000円以上 |
| スマホスタンド | 300〜500円 | 1,000円以上 |
| USB充電器(急速) | 500〜800円 | 1,500円以上 |
| LEDライト | 300〜800円 | 1,500円以上 |

**注意**: 安すぎる場合は品質問題や容量偽装の可能性あり

### 2.4 販売元・出品者の特徴

#### 怪しい販売元の特徴
- 電話番号が「+86」（中国国番号）で始まる
- 住所の末尾に「CN」がある
- 住所に中国の都市名（深圳、广州、东莞など）
- 会社名が日本風だが所在地が海外

#### 確認すべきポイント
1. 販売元をクリックして詳細確認
2. 「販売元: Amazon.co.jp」が最も安心
3. 日本国内の法人かどうか

### 2.5 商品画像の特徴

#### 怪しい画像パターン
- 雷や電気のエフェクト（特にイヤホン・充電器）
- 過度に加工された光沢感
- 背景が派手すぎる
- 比較画像で他社を貶める表現

### 2.6 レビューの特徴

#### サクラレビューの兆候
- 短期間にレビューが急増
- 極端に高評価（4.5以上）が多い
- レビュー内容が似通っている
- 商品と関係ない内容のレビュー
- 日本語が不自然

## 3. 現在のフィルタリングロジック評価

### 3.1 現在実装済みの機能

#### constants.js
```javascript
// 海外発送キーワード
OVERSEAS_SHIPPING_KEYWORDS = ['海外', 'China', '中国', '2-4週間', '3-4週間']
```

#### seller-checker.js
```javascript
// 日本の都道府県判定
JAPAN_PREFECTURES = ['北海道', '東京', '大阪', ...]

// 海外インジケーター
OVERSEAS_INDICATORS = ['中国', 'China', 'CN', '深圳', '广东', ...]
```

#### filter-utils.js
```javascript
// フィルタープリセット
- standard: Prime対象 (p_85:2322926051)
- premium: Prime + 評価4以上
- strict: Amazon公式販売のみ (p_6:AN1VRQENFRJN5)
```

### 3.2 現在検出できるもの

| 項目 | 検出可否 | 実装状況 |
|-----|---------|---------|
| 海外セラー（住所ベース）| ○ | seller-checker.js |
| Prime非対応商品 | ○ | URLパラメータ |
| 評価4未満 | ○ | URLパラメータ |
| Amazon公式以外 | ○ | URLパラメータ |
| 海外発送キーワード | ○ | constants.js |

### 3.3 現在検出できないもの

| 項目 | 検出可否 | 理由 |
|-----|---------|-----|
| 怪しいブランド名 | × | パターン未実装 |
| タイトルの誇張表現 | × | パターン未実装 |
| 同一商品の複数出品 | × | 画像比較が必要 |
| サクラレビュー | × | 外部API連携が必要 |
| 価格の異常性 | × | カテゴリ別相場データが必要 |
| 商品画像のエフェクト | × | 画像解析が必要 |

const SUSPICIOUS_BRAND_PATTERNS = {

### 4.4 サクラチェッカー連携（提案）

外部サービス「サクラチェッカー」の判定基準を参考に、以下をチェック:

1. **ショップ情報**: 発送地域、電話番号
2. **レビュー分布**: 不自然な5つ星の集中
3. **レビュー日付**: 短期間での急増
4. **レビュー本文**: テンプレート的な内容

## 5. カテゴリ別分析

### 5.1 スマホスタンド

**転売品/OEM品の特徴**
- 形状がほぼ同一で複数ブランドが販売
- 価格帯: 300〜1,500円が激戦区
- 素材表記が曖昧（「高品質素材」など）

**信頼できる商品の特徴**
- 日本メーカー（ELECOM、BUFFALO等）
- 具体的な素材記載（アルミニウム合金等）
- 技適マーク表示（充電機能付きの場合）

### 5.2 モバイルバッテリー

**転売品/OEM品の特徴**
- 容量偽装の可能性（実際より大容量を表示）
- 25000mAhで2,500円などの異常価格
- PSEマーク表示なし/偽装

**信頼できる商品の特徴**
- Anker、cheero、RAVPower等の実績あるブランド
- PSEマーク認証済み
- 容量に対して適正価格

### 5.3 ワイヤレスイヤホン

**転売品/OEM品の特徴**
- 「ノイズキャンセリング」を謳うが効果なし
- AirPods似のデザイン（著作権侵害の可能性）
- Bluetooth5.3など最新規格を強調

**信頼できる商品の特徴**
- 技適マーク取得
- 有名オーディオブランド
- 試聴レビューで音質評価あり

### 5.4 USB充電器

**転売品/OEM品の特徴**
- 過大なW数表記
- 認証マーク（PSE/UL）なし
- 発熱・発火リスク

**信頼できる商品の特徴**
- PSEマーク認証済み
- 有名メーカー（Anker、Belkin等）
- 適正な価格設定

### 5.5 LEDライト

**転売品/OEM品の特徴**
- ルーメン値の誇大表記
- 「●万時間寿命」などの非現実的表記
- 写真撮影用リングライトに多い

**信頼できる商品の特徴**
- 日本メーカー
- 具体的なスペック記載
- 保証期間の明記

## 6. 改善提案

### 6.1 短期的改善（実装容易）

1. **怪しいブランド名パターンの検出**
   - 正規表現によるパターンマッチング
   - 警告バッジの表示

2. **タイトル誇張表現の検出**
   - キーワードリストによるマッチング
   - スコアリングシステム

3. **価格異常の警告**
   - カテゴリ別閾値の設定
   - 極端に安い場合に警告

### 6.2 中期的改善（要開発）

1. **複合スコアリングシステム**
   - ブランド名 + タイトル + 価格 + セラー情報
   - 総合スコアで判定

2. **ホワイトリスト/ブラックリスト**
   - 信頼できるブランドのホワイトリスト
   - 既知の怪しいブランドのブラックリスト

### 6.3 長期的改善（要外部連携）

1. **サクラチェッカーAPI連携**
   - レビューの信頼性スコア取得

2. **同一商品検出**
   - 画像ハッシュによる類似商品検出
   - 同一商品の複数ブランド販売を警告

## 7. 結論

### 現在のロジックでの検出率

| 項目 | 推定検出率 |
|-----|----------|
| 海外セラー（住所ベース）| 70-80% |
| Prime非対応商品 | 100% |
| 低評価商品 | 100% |
| 怪しいブランド名 | 0% |
| 誇張タイトル | 0% |
| サクラレビュー | 0% |

### 推奨対応優先度

1. **高優先度**: 怪しいブランド名パターン検出
2. **高優先度**: タイトル誇張表現検出
3. **中優先度**: 価格異常警告
4. **中優先度**: ホワイトリスト機能
5. **低優先度**: 外部API連携
